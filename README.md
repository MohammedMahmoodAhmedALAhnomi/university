# نظام اللجنة العلمية لإدارة المحتوى التعليمي

نظام متكامل لإدارة المحتوى التعليمي للجان العلمية بالجامعات، مبني باستخدام PHP (MVC) مع Bootstrap 5 و RTL.

---

## المميزات

### الواجهة الأمامية (الموقع العام)
- عرض التخصصات والمستويات والفصول الدراسية
- تصفح المواد الدراسية وتفاصيلها
- تحميل الملفات التعليمية (محاضرات، ملخصات، نماذج اختبارات)
- تقييم المواد الدراسية بنظام 5 نجوم
- بحث متقدم في المواد مع فلترة
- إعلانات عامة مع إمكانية التثبيت
- صفحة نبذة عن النظام
- واجهة متجاوبة (Responsive) مع دعم الوضع الليلي

### لوحة التحكم (Admin Panel)
- لوحة إحصائيات متكاملة مع رسوم بيانية
- إدارة التخصصات (إضافة/تعديل/حذف)
- إدارة المستويات الدراسية
- إدارة الفصول الدراسية
- إدارة المواد الدراسية
- رفع وإدارة الملفات مع تصنيفها
- إدارة المستخدمين والأدوار
- إدارة المشرفين وتحديد صلاحياتهم
- إدارة الإعلانات مع جدولة النشر
- إعدادات الموقع (المظهر، الروابط الاجتماعية، الملفات)

### نظام الأدوار والصلاحيات
| الدور | الصلاحيات |
|-------|-----------|
| **Admin** | وصول كامل لكل المزايا والإعدادات |
| **Manager** | صلاحيات محدودة حسب التخصص/المستوى المعين |
| **Guest** | تصفح الموقع العام فقط |

- **PHP** >= 7.4
- **MySQL** >= 5.7
- **Apache** مع mod_rewrite
- **الإضافات المطلوبة**: PDO, mbstring

| المكتبة | الإصدار | الاستخدام |

## التثبيت والتشغيل

### 1. تشغيل محلياً (XAMPP)

```bash
# انسخ المشروع إلى مجلد htdocs
# شغل Apache و MySQL من XAMPP Control Panel
```

### 2. إنشاء قاعدة البيانات
- افتح phpMyAdmin: http://localhost/phpmyadmin
- استورد ملف `database/schema.sql`
- أو نفذ الأوامر يدوياً

### 3. إعداد ملف البيئة
انسخ الملف `.env` وعدل الإعدادات التالية:
```
APP_URL=http://localhost/university_system
DB_NAME=university_system
DB_USER=root
DB_PASS=
```

### 4. تشغيل Composer (لتحميل المكتبات)
```bash
cd C:\xampp\htdocs\university_system
composer install
```

### 5. الوصول للموقع
```
الموقع العام:  http://localhost/university_system
لوحة التحكم:   http://localhost/university_system/login
بريد المدير:   admin@university.edu
كلمة السر:     password
```

### الرفع على استضافة مجانية (000WebHost)
راجع دليل الرفع المرفق في ملفات التوثيق.

## قاعدة البيانات

| الجدول | الوصف |
|--------|-------|
| `majors` | التخصصات العلمية |
| `levels` | المستويات الدراسية |
| `semesters` | الفصول الدراسية |
| `courses` | المواد الدراسية (مرتبطة بتخصص + مستوى + فصل) |
| `users` | المستخدمون (admin/manager/guest) |
| `files` | الملفات التعليمية (مرفوعة مع تصنيف) |
| `announcements` | الإعلانات |
| `ratings` | تقييمات المواد (1-5 نجوم) |
| `sessions` | جلسات المستخدمين |
| `settings` | إعدادات الموقع |
| `activity_logs` | سجل النشاطات |

## المسارات (Routes)

### عامة (بدون تسجيل)
| الطريقة | المسار | الوظيفة |
|---------|--------|---------|
| GET | `/` | الصفحة الرئيسية |
| GET | `/courses/{id}` | تفاصيل المادة |
| GET | `/majors/{id}` | تفاصيل التخصص |
| GET | `/search` | البحث |
| GET | `/announcements` | الإعلانات |

| الطريقة | المسار | الوظيفة |
|---------|--------|---------|

## الملفات المرفوعة

- جميع الملفات تُرفع إلى `public/uploads/files/`
- صور الإعدادات تُرفع إلى `public/uploads/settings/`
- الوصول المباشر لمجلد `uploads/` محظور عبر `.htaccess`
- يتم تقديم الملفات عبر `FileController` (download/preview)
- أقصى حجم للملف: 10 ميجابايت (قابل للتعديل)
- الامتدادات المسموحة: pdf, doc, docx, ppt, pptx, xls, xlsx, zip, rar
